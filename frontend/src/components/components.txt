components : 재사용 가능한 UI 구성요소


✅ src/components/ 폴더에 들어갈 수 있는 예시
컴포넌트	설명
Nav.jsx	상단 네비게이션 바
Footer.jsx	하단 푸터 영역
Button.jsx	커스텀 버튼 컴포넌트
Card.jsx	게시글/상품 등을 감싸는 카드 컴포넌트
Modal.jsx	공통 팝업/모달 창
LoadingSpinner.jsx	로딩 중 표시되는 스피너
Pagination.jsx	게시판이나 목록에서 페이지네이션 컴포넌트
Input.jsx, FormGroup.jsx	커스텀 입력 필드 또는 폼 요소

👆 이런 컴포넌트들은 어떤 페이지에서든 재사용 가능해야 하므로 components 폴더에 두는 게 일반적입니다.
